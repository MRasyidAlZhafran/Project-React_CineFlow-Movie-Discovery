# CineFlow

A minimalist and premium movie discovery web application built with **React** and **TVMaze API**.

## Features
- **Dynamic Search**: Instant search with URL state persistence.
- **Premium UI**: Dark mode design with glassmorphism effects.
- **Detailed Insights**: Immersive show details and metadata.
- **Responsive**: Fully optimized for various screen sizes.

## Tech Stack
- React (Vite)
- React Router DOM
- Lucide Icons
- Vanilla CSS

## Quick Start
1. **Clone & Install**:
   ```bash
   git clone <repo-url>
   cd <repo-folder>
   npm install
   ```
2. **Run**:
   ```bash
   npm run dev
   ```
